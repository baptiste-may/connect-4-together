# Politique de Confidentialité

## Collecte des Données

Le site "Connect 4 Together" ne collecte aucune information personnelle des utilisateurs. Les seules données stockées sont le token de reconnexion et le nom d'utilisateur, qui sont enregistrés dans le navigateur de l'utilisateur.

## Utilisation des Données

Les données stockées dans le navigateur sont utilisées uniquement pour permettre la reconnexion automatique des utilisateurs et pour partager le nom d'utilisateur choisi.

## Stockage des Données

Les données sont stockées localement dans le navigateur de l'utilisateur via le localStorage. Aucune donnée personnelle n'est stockée sur nos serveurs.

## Google Analytics

Nous utilisons Google Analytics pour analyser la fréquentation du site. Les données collectées par Google Analytics sont anonymisées et utilisées uniquement à des fins statistiques. Elles sont conservées pendant une durée de 14 mois, conformément aux recommandations du RGPD.

## Droits des Utilisateurs

Conformément à la réglementation en vigueur, les utilisateurs disposent des droits suivants :
- Droit d'accès : vous pouvez demander à accéder à vos données personnelles.
- Droit de rectification : vous pouvez demander la correction de vos données personnelles si elles sont inexactes.
- Droit à l'effacement : vous pouvez demander la suppression de vos données personnelles.
- Droit d'opposition : vous pouvez vous opposer au traitement de vos données personnelles.

Pour exercer ces droits, veuillez nous contacter à l'adresse suivante : [pro@may-baptiste.fr](mailto:pro@may-baptiste.fr).

## Modifications de la Politique de Confidentialité

Nous nous réservons le droit de modifier cette politique de confidentialité à tout moment. Les modifications seront publiées sur cette page.

## Contact

Pour toute question concernant notre politique de confidentialité, veuillez nous contacter à l'adresse suivante : [pro@may-baptiste.fr](mailto:pro@may-baptiste.fr).
